# Job Search Automation Engine

Automated pipeline that scrapes daily job postings, tailors a base CV using AI, and drafts cover letters.

## Features
- Daily job scraping via SerpAPI (Google Jobs).
- AI-powered CV tailoring and cover letter generation.
- Email delivery of tailored application materials.
- Fully automated with GitHub Actions.

## Setup
1. Clone the repo.
2. Install dependencies: `uv sync`.
3. Set up `.env` with your API keys.
4. Run `uv run python main.py`.

See `roadmap.md` for the full development plan.
